# Arpit Jaiswal - Portfolio

Personal portfolio website built with React, Vite, Tailwind CSS, and Lucide React. 

## Features
- **About Me**: Professional backstory and mission statement.
- **Projects & Work**: Details of shipped products and case studies.
- **Skills**: Toolkit and areas of expertise.
- **Direct Resume Download**: Fast access to PDF resume.
- **Contact Channels**: Integrated contact form via Formspree.

## Getting Started

### Run Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```
